# VCard Personal Template
A personal vCard Template built for fun.. ;)

Built with <b>Gatsby, React, Styled Component, Material UI...</b>

<b>To Install and run the website:</b>

  ```bash
  yarn
  yarn build
  yarn start
  ```

<b>To develop from template:</b>

  ```bash
  yarn
  yarn develop
  ```
